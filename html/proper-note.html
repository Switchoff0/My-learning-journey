Beginner HTML Notes

Notes for reference

Imp Keyword

  • tag

    There are two type of tages

    • Opening Tag

      • Eg.<h1>,<p>.

    • Closing Tags

      • Eg.</h1>,</p>.

  • element

    Eg.<h1>Hello World</h1>

  • Void Element

    Void element does not have the closing tag and it does not have any content.

    Eg.<img />,<meta />.

  • Attribute

    An attribute is a value placed inside the opening tag of HTML element. Attributes provide additional information about the element or specify how the element should behave.

    Eg.<element attribute ="value" ></element>

    Example of different attribute for different element

    • src,altattribute for img element.
    • href,targetattribute for anchor element

  • boolean attribute

Basic Element

Commonly used element

  • Heading element

    <h1>hello world</h1>

  • Paragraph element

    <p>this is paragraph element</p>

  • image element

    This is Void element.

    src attributeis used for source of image.

    alt attribute is used to display alternative text, for blind and to show text if img failed to load

    Eg.

    • <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="two kitten" />

  • Anchor element

    Anchor element is used to create hyperlink(to show clickable link).

    href attribute is used to refer link

    target="_blank" attribute is used to open link in new page in browser

    Eg.

    • <a href="https://www.freecodecamp.org/news/" target="_blank" >visit freecodecamp</a>
    • preview of anchor element;visit freecodecamp

  • input element

  • Comment element
